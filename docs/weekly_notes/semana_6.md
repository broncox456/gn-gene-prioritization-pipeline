# Semana 6

## Objetivo
Consolidar el pipeline de análisis y trabajar con un dataset estructurado con características similares a datos reales de GEO.

## Actividades realizadas
- Se evaluó la integración de datos externos provenientes de fuentes públicas.
- Se decidió trabajar con un dataset estructurado tipo GEO para garantizar reproducibilidad.
- Se ejecutó el pipeline completo utilizando este dataset.
- Se verificó la consistencia del análisis, validación y visualización.

## Resultados
El pipeline es capaz de procesar datos estructurados de forma consistente, manteniendo coherencia en los resultados y visualización.

## Observaciones
El uso de datasets estructurados permite mantener control sobre la calidad de los datos y facilita la reproducibilidad del análisis.

## Dificultades encontradas
- Problemas de acceso a fuentes externas en tiempo real.
- Dependencia de URLs inestables para descarga de datos.

## Conclusión
Se consolidó un pipeline reproducible, funcional y adaptable a datos reales, estableciendo una base sólida para futuros análisis con datasets completos de GEO.
