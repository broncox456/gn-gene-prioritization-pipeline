# Semana 3

## Objetivo
Incorporar una visualización básica para facilitar la interpretación de los genes priorizados.

## Actividades realizadas
- Se instaló la librería matplotlib para la generación de gráficos.
- Se desarrolló un script en Python para visualizar los resultados del pipeline.
- Se seleccionaron los 5 genes con mayor puntuación de prioridad.
- Se generó una gráfica de barras representando los genes priorizados.
- Se guardó la figura en formato PNG dentro de la carpeta `results/figures`.

## Resultados
Se obtuvo una representación visual clara de los genes más relevantes, lo que mejora la interpretación de los resultados del pipeline.

## Observaciones
Los genes IL6, NPHS1 y COL4A4 destacan visualmente como los más relevantes, lo cual es coherente con procesos inflamatorios y daño glomerular.
