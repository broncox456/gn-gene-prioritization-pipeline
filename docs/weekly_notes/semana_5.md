# Semana 5

## Objetivo
Incorporar un dataset con características más cercanas a datos reales reportados en GEO.

## Actividades realizadas
- Se generó un dataset basado en genes reportados en la literatura sobre glomerulonefritis.
- Se añadieron nuevos genes relevantes como MMP9, TGFB1 y VEGFA.
- Se actualizó el pipeline para trabajar con este nuevo dataset.
- Se ejecutó nuevamente todo el flujo de análisis incluyendo validación y visualización.

## Resultados
El pipeline logró procesar correctamente un dataset más complejo, manteniendo consistencia en los resultados y coherencia biológica.

## Observaciones
Los genes priorizados muestran relevancia en inflamación, fibrosis y daño glomerular, lo cual fortalece la validez clínica del análisis.

## Dificultades encontradas
- Ajuste del pipeline para asegurar compatibilidad con nuevos datos.
- Verificación de coherencia biológica de los resultados.

## Próximo paso
Descargar y procesar datos reales directamente desde GEO.
