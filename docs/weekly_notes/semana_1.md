# Semana 1

## Objetivo
Desarrollar la primera versión funcional de un pipeline de priorización de genes en glomerulonefritis.

## Actividades realizadas
- Se creó la estructura inicial del proyecto organizando carpetas para datos, scripts, resultados y documentación.
- Se diseñó un dataset de prueba (toy dataset) con genes relevantes en nefrología y procesos inflamatorios.
- Se desarrolló un script en Python para:
  - leer datos de expresión génica
  - clasificar genes según significancia
  - calcular una puntuación de prioridad
- Se generó un archivo de salida con los genes ordenados por relevancia.

## Resultados
Se obtuvo una primera versión funcional del pipeline capaz de procesar datos y generar resultados reproducibles.

## Dificultades encontradas
- Problemas iniciales con la ejecución de scripts en PowerShell.
- Confusión en rutas de archivos dentro del proyecto.
- Necesidad de verificar la correcta estructura del proyecto.

## Próximo paso
Mejorar la estabilidad del pipeline y añadir validación estructural de los resultados.
